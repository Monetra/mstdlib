72 /*! Find the appropriate layer and grab the handle and lock it.
73  * \param[in] io Pointer to io object
74  * \param[in] layer_id id of layer to lock, or M_IO_LAYER_FIND_FIRST_ID to search for layer.
75  * \param[in] name Name of layer to lock. This can be used as a sanity check to ensure
76  * the layer id really matches the layer type. Use NULL if name matching
77  * is not required. If M_IO_LAYER_FIND_FIRST_ID is used for the layer_id,
78  * this parameter cannot be NULL.
79  * \return locked io layer, or NULL on failure
