typedef M_http_error_t(* M_http_reader_header_func)(const char *key, const char *val, void *thunk)

Function definition for reading headers.

Headers are split if a header list. Keys will appear multiple times if values were in a list or if the header appears multiple times. Values with semicolon (;) separated parameters are not split.
