Detailed Description

Generic table construction.

Notes:

  • Headers optional.
  • Expand flag will allow adding data without setting explicit table size.

enum M_table_insert_flags_t
Enumerator
M_TABLE_INSERT_NONE 

Fail the insert if the header, or index does not exist.

M_TABLE_INSERT_EXPAND 

Add a named column if it does not exist.

M_TABLE_INSERT_IGNORECOLS 

Ignore names if a corresponding named header does not exist.

enum M_table_flags_t

Default operation.
