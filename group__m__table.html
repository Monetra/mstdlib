Detailed Description

Generic table construction.

Notes:

  • Headers optional.
  • Expand flag will allow adding data without setting explicit table size.

Enumeration Type Documentation

enum M_table_insert_flags_t
Enumerator
M_TABLE_INSERT_NONE 

Fail the insert if the header, or index does not exist.

M_TABLE_INSERT_EXPAND 

Add the header if it does not exist. If by index, add cell and any empty cells before the index as needed.

M_TABLE_INSERT_IGNOREHEADER 

Ignore the header if it does not exist.
